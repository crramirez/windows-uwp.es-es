# <a name="cortana-interactions-in-windows-apps"></a>Interacciones de Cortana en aplicaciones de Windows

Cortana ofrece un marco de extensibilidad completo y sólido que permite incorporar perfectamente la función de tu aplicación o servicio en la experiencia de Cortana.

## <a name="weve-moved"></a>Nos hemos mudado

Toda la documentación de desarrolladores de servicios y características de Cortana está disponible ahora a través del [Centro de desarrollo de Cortana](https://developer.microsoft.com/cortana).

Para empezar, consulte la [Introducción a Cortana Skills kit](https://docs.microsoft.com/cortana/skills/overview).

Para obtener información sobre cómo ampliar Cortana con la funcionalidad de una aplicación de Windows mediante comandos de voz, consulte [comandos de voz de Cortana](https://docs.microsoft.com/cortana/voice-commands/vcd).
