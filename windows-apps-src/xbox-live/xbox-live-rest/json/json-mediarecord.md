# <a name="mediarecord-json"></a>MediaRecord (JSON)
 
<a id="ID4EO"></a>

 
## <a name="mediarecord"></a>MediaRecord
 
El objeto MediaRecord tiene la especificación siguiente.
 
| Miembro| Tipo| Descripción| 
| --- | --- | --- | 
| id| string| Identificador de los medios utilizados por el catálogo de Bing o el catálogo de proveedor.| 
| idType| string| Cómo interpretar el identificador de medios. Los valores posibles incluyen "bing" y "provider".| 
| name| string| Nombre localizado del contenido multimedia.|
