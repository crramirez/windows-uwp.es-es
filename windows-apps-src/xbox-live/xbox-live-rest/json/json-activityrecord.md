# <a name="activityrecord-json"></a>ActivityRecord (JSON)
Una cadena formateada y localizada sobre presencia enriquecida de uno o varios usuarios. 
<a id="ID4EN"></a>

 
## <a name="activityrecord"></a>ActivityRecord
 
El objeto ActivityRecord tiene la especificación siguiente.
 
| Miembro| Tipo| Descripción| 
| --- | --- | --- | 
| richPresence| string| La cadena de presencia enriquecida, con formato y localizados.| 
| Media| MediaRecord| Lo que el usuario está viendo o escuchar.| 
  
<a id="ID4ETB"></a>

 
## <a name="sample-json-syntax"></a>Sintaxis de JSON de ejemplo
 

```json
{
        richPresence:"Team Deathmatch on Nirvana"
      }
    
```
