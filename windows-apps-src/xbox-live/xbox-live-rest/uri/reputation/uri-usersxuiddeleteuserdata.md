# <a name="usersxuidxuiddeleteuserdata"></a>/users/xuid({xuid})/deleteuserdata
Restablece completamente los datos de reputación de un usuario de prueba. Solo para pruebas. Es el dominio para estos URI `reputation.xboxlive.com`. Este URI siempre se llama en el puerto 10443.
 
  * [Parámetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parámetros de URI
 
| Parámetro| Tipo| Descripción| 
| --- | --- | --- | 
| xuid| entero sin signo de 64 bits| Id. de usuario de Xbox (XUID) del usuario cuyos datos se está eliminando.|
