# <a name="usersxuidxuidfeedback"></a>/users/xuid({xuid})/feedback
 
Es el dominio para estos URI `reputation.xboxlive.com`.
 
<a id="ID4EW"></a>

 
## <a name="uri-parameters"></a>Parámetros de URI
 
| Parámetro| Tipo| Descripción| 
| --- | --- | --- | 
| xuid| ulong| Id. de usuario de Xbox (XUID) del usuario que se va a notificar.| 
  
<a id="ID4EZB"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[POST](uri-reputationusersxuidfeedbackpost.md)

&nbsp;&nbsp;Usar desde el título si así lo desea agregar una opción de comentarios en el juego, en lugar de usar el shell.
