# <a name="usersxuidxuidoutbox"></a>/users/xuid({xuid})/outbox
Proporciona acceso de solo envío a un usuario de mensajería de bandeja de salida para los servicios de Xbox LIVE. Es el dominio para estos URI `msg.xboxlive.com`.
 
  * [Parámetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parámetros de URI 
 
| Parámetro| Tipo| Descripción| 
| --- | --- | --- | 
| xuid | entero de 64 bits sin signo | La consola Xbox usuario ID (XUID) del Reproductor que realiza la solicitud. | 
  
<a id="ID4EXB"></a>

 
## <a name="valid-methods"></a>Métodos válidos 

[POST (/users/xuid({xuid})/outbox)](uri-usersxuidoutboxpost.md)

&nbsp;&nbsp;Envía un mensaje especificado a una lista de destinatarios.
