# <a name="leaderboards-uris"></a>URI de tablas de clasificación

Esta sección proporciona información detallada sobre las direcciones de identificador de recursos Universal (URI) y los métodos de protocolo de transporte de hipertexto (HTTP) asociados desde servicios de Xbox Live para *marcadores*.

El dominio para estos URI es leaderboards.xboxlive.com.

<a id="ID4EDB"></a>


## <a name="in-this-section"></a>En esta sección

[/scids/{scid}/leaderboards/{leaderboardname}](uri-scidsscidleaderboardsleaderboardname.md)

&nbsp;&nbsp;Tiene acceso a un tablero de puntuaciones global predefinido.

[/users/xuid({xuid})/scids/{scid}/stats/{statname)/people/{all\|favorite}](uri-usersxuidscidstatnamepeople.md)

&nbsp;&nbsp;Tiene acceso a un marcador social (clasificación).
