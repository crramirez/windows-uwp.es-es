# <a name="usersmegroupsmoniker"></a>/users/me/groups/{moniker}
Tiene acceso a la PresenceRecord para mi grupo. Es el dominio para estos URI `userpresence.xboxlive.com`.
 
  * [Parámetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parámetros de URI
 
| Parámetro| Tipo| Descripción| 
| --- | --- | --- | 
| moniker| string| Cadena que define el grupo de usuarios. El moniker solo aceptado en la actualidad es "People", con una letra mayúscula "P".| 
  
<a id="ID4ERB"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[GET (/users/me/groups/{moniker} )](uri-usersmegroupsmonikerget.md)

&nbsp;&nbsp;Obtiene el PresenceRecord para mi grupo.
