# <a name="usersxuidxuid"></a>/users/xuid({xuid})
Obtener acceso a la presencia de otro usuario o cliente. Es el dominio para estos URI `userpresence.xboxlive.com`.
 
  * [Parámetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parámetros de URI
 
| Parámetro| Tipo| Descripción| 
| --- | --- | --- | 
| xuid| entero sin signo de 64 bits| Id. de usuario de Xbox (XUID) del usuario de destino.| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[GET (/users/xuid({xuid}))](uri-usersxuidget.md)

&nbsp;&nbsp;Detectar la presencia de otro usuario o cliente.
