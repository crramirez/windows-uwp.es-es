# <a name="usersmeinventory"></a>/users/me/inventory
Obtiene acceso al conjunto de inventario actualmente asociado con el usuario proporcionado. Es el dominio para estos URI `inventory.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[GET (/users/me/inventory)](uri-inventoryget.md)

&nbsp;&nbsp;Proporciona el conjunto de inventario actualmente asociado con el usuario proporcionado al llamador.
