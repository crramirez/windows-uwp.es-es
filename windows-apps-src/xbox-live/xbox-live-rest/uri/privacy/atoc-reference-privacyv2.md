# <a name="privacy-uris"></a>URI de privacidad
 
Esta sección proporcionan detalles sobre las direcciones de identificador de recursos Universal (URI) y los métodos de protocolo de transporte de hipertexto (HTTP) asociados de servicios de Xbox Live para *privacidad*.
 
Solo los juegos que se ejecutan en Windows 8, en una consola Xbox 360 o un Windows Phone dispositivo puede usar este servicio.
 
El dominio para estos URI es privacy.xboxlive.com.
 
<a id="ID4EPB"></a>

 
## <a name="in-this-section"></a>En esta sección

[/users/{ownerId}/people/mute](uri-privacyusersowneridpeoplemute.md)

&nbsp;&nbsp;Tiene acceso a la lista de silencio de un usuario.

[/users/{requestorId}/permission/validate](uri-privacyusersrequestoridpermissionvalidate.md)

[/users/{ownerId}/people/avoid](uri-privacyusersxuidpeopleavoid.md)

&nbsp;&nbsp;Tiene acceso a la lista evite para un usuario
