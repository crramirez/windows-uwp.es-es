# <a name="usersxuidxuidlistspinslistnameremoveitems"></a>/users/xuid(xuid)/lists/PINS/{listname}/RemoveItems
Quita elementos de una lista. Es el dominio para estos URI `eplists.xboxlive.com`.
 
  * [Parámetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parámetros de URI 
 
| Parámetro| Tipo| Descripción| 
| --- | --- | --- | 
| XUID| string| XUID del usuario.| 
| nombre de lista| string| Nombre de la lista para manipular.| 
  
<a id="ID4E5B"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[ELIMINAR](uri-usersxuidlistspinslistnameremoveitemsdelete.md)

&nbsp;&nbsp;Quita los elementos de una lista por índice.

[POST](uri-usersxuidlistspinslistnameremoveitemspost.md)

&nbsp;&nbsp;Quita elementos de una lista por Id.
