# <a name="usersxuidxuidlistspinslistnamecontainsitems"></a>/users/xuid(xuid)/lists/PINS/{listname}/ContainsItems
Determina si un conjunto de elementos (especificado por ID) se encuentran en una lista sin tener que recuperar toda la lista. Es el dominio para estos URI `eplists.xboxlive.com`.
 
  * [Parámetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parámetros de URI 
 
| Parámetro| Tipo| Descripción| 
| --- | --- | --- | 
| XUID| string| XUID del usuario.| 
| nombre de lista| string| Nombre de la lista para manipular.| 
  
<a id="ID4E5B"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[POST](uri-usersxuidlistspinslistnamecontainsitemspost.md)

&nbsp;&nbsp;Determina si una lista contiene un conjunto de elementos (especificado por itemId) sin tener que recuperar toda la lista.
