# <a name="user-statistics-uris"></a>URI de estadísticas de usuario
 
Esta sección proporciona información detallada sobre las direcciones de identificador de recursos Universal (URI) y los métodos de protocolo de transporte de hipertexto (HTTP) asociados desde servicios de Xbox Live para *estadísticas de usuario*.
 
El dominio para estos URI es userstats.xboxlive.com.
 
<a id="ID4EDB"></a>

 
## <a name="in-this-section"></a>En esta sección

[/batch](uri-batch.md)

&nbsp;&nbsp;Tiene acceso a las solicitudes por lotes complejos para varias estadísticas del Reproductor a través de varios títulos.

[/users/xuid({xuid})/scids/{scid}/stats](uri-usersxuidscidsscidstats.md)

&nbsp;&nbsp;Tiene acceso a una configuración de servicio con ámbito de una lista delimitada por comas de nombres de estadísticas de usuario en nombre del usuario especificado.
