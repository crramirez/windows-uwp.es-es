# <a name="batch"></a>/batch
Tiene acceso a las solicitudes por lotes complejos para varias estadísticas del Reproductor a través de varios títulos. Es el dominio para estos URI `userstats.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[POST](uri-batchpost.md)

&nbsp;&nbsp;Método que funciona como un método GET para las solicitudes por lotes complejos para varias estadísticas del Reproductor a través de varios títulos de POST.
