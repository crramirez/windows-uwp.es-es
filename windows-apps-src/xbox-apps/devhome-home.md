# <a name="home-page-dev-home"></a>Página principal (desarrollo principal)
   
  
La página **principal** de desarrollo principal proporciona acceso rápido a las tareas que se realizan habitualmente en una consola de desarrollo.   
 ![Página principal de la página principal de desarrollo](images/devhome_home.png)   
  
La página **principal** contiene los paneles siguientes:   
 
   *  [Quick actions](#ID4EEB)  
   *  [Juegos y aplicaciones](#ID4EPC)  
   *  [Cuentas de prueba](#ID4EQD)  
   *  [Conectar con el explorador](#ID4EFE)  

 
<a id="ID4EEB"></a>

   

## <a name="quick-actions"></a>Quick actions  
   
  
El panel de **acciones rápidas** contiene los siguientes botones que proporcionan acceso rápido a las acciones que se realizan con frecuencia:   
 
   *  **Pin mostrar Visual Studio:** Muestra un código que puede usar para emparejar la consola de desarrollo con la instalación de Visual Studio en el equipo.   
   *  **Espacio aislado del cambio:** Permite cambiar qué espacio aislado es la consola de.   
   *  **Consola de reinicio:** Reinicia la consola de desarrollo.   
   *  **Salir del modo de desarrollo:** Devuelve la consola al modo de venta por menor y se reinicia la consola.   
   *  **Inicio principal:** Se abre la pantalla de inicio de venta por menor.   
   *  **Registrar un juego desde una ubicación de red compartida:** Registra un juego desde una ubicación de red compartida.   

  
<a id="ID4EPC"></a>

   

## <a name="games--apps"></a>Juegos y aplicaciones   
   
  
El panel **juegos y aplicaciones** puede usarse para llevar a cabo las siguientes acciones:   
 
   *  Ver la lista de juegos y aplicaciones instaladas en la consola de desarrollo.  
   *  Iniciar rápidamente juegos y aplicaciones.  
   *  Ver el estado PLM de juegos y aplicaciones.  
   *  Cambiar los Estados PLM desde un menú contextual.  
   *  Registrar y ejecutar un título desde un recurso compartido de red remoto.

  
<a id="ID4EQD"></a>

   

## <a name="test-accounts"></a>Cuentas de prueba  
   
  
El panel de **cuentas de prueba** puede utilizarse para realizar las siguientes acciones:   
 
   *  Ver la lista de usuarios registrados en la consola.  
   *  Inicie la sesión de un usuario o alejar con un solo clic.  
   *  Agregue los usuarios y las cuentas de invitado.  
   *  Ver detalles de los usuarios y las cuentas de invitado.  

  
<a id="ID4EFE"></a>

   

## <a name="connect-with-your-browser"></a>Conectar con el explorador  
   
  
El panel de **Conectar con el explorador** muestra la dirección IP y el puerto en el que puede usar para administrar la consola desde un explorador web.
