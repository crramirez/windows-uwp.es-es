# <a name="networking-page-dev-home"></a>Página de red (desarrollo principal)
   
  
La página de principal de desarrollo de **redes** simula diversas condiciones de redes para solucionar problemas. También permite iniciar un seguimiento de paquetes de red.   
 ![Página de red de la página principal de desarrollo](images/devhome_networking.png)   
  
La página de **redes** contiene los paneles siguientes:   
 
   *  [Simulación de red](#ID4EEB)  
   *  [XBTrace (netcap)](#ID4EOB)  

 
<a id="ID4EEB"></a>

   

## <a name="network-simulation"></a>Simulación de red  
   
  
El panel de **simulación de red** ejecuta una simulación de red que probar frente a varios perfiles de ancho de banda.   
  
<a id="ID4EOB"></a>

   

## <a name="xbtrace-netcap"></a>XBTrace (netcap)  
   
  
El panel **XBTrace (netcap)** se inicia o detiene un seguimiento de paquetes de red.   
  
<a id="ID4E2B"></a>
