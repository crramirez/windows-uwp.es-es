# <a name="whats-new-in-windows-10-for-developers-build-17763"></a>Novedades de Windows 10 para desarrolladores, compilación 17763

Compilación de Windows 10 17763 (también conocido como el de octubre de 2018 actualización o versión 1809), junto con Visual Studio 2017 y el SDK actualizado, proporciona las herramientas, características y experiencias para que notables aplicaciones de plataforma Universal de Windows. [Instala las herramientas y el SDK](https://go.microsoft.com/fwlink/?LinkId=821431) en Windows 10 y estarás listo para [crear una nueva aplicación universal de Windows](../get-started/create-uwp-apps.md) o para explorar cómo puedes usar tu [código de aplicación existente en Windows](../porting/index.md).

A continuación te ofrecemos una colección de características e instrucciones nuevas y mejoradas de interés para los desarrolladores de Windows en esta versión. Para obtener una lista completa de los espacios de nombres nuevo agregado para el SDK de Windows, consulte el [17763 cambios en la API de generación de Windows 10](windows-10-build-17763-api-diff.md). Para obtener más información sobre las características más destacadas de Windows 10, consulta [Lo bueno de Windows 10](https://go.microsoft.com/fwlink/?LinkId=823181). También puedes consultar [Funcionalidades de la plataforma de desarrolladores de Windows](https://developer.microsoft.com/windows/platform/features) si quieres tener una descripción general de alto nivel de las funcionalidades pasadas y futuras que se han agregado en la plataforma de Windows.

Herramienta de empaquetado MSIX | El nuevo [herramienta de empaquetado MSIX](https://docs.microsoft.com/windows/msix/mpt-overview)) le permite volver a empaquetar las aplicaciones de escritorio existentes en el formato MSIX, incluso si no tiene acceso a su código fuente. Se puede ejecutar en la línea de comandos o a través de su interfaz de usuario interactiva.
Soporte técnico de Desktop App Converter para MSIX | Puede usar el [Desktop App Converter](https://docs.microsoft.com/windows/uwp/porting/desktop-to-uwp-root) para generar un paquete MSIX, utilizando el `-MakeMSIX` parámetro.
Compatibilidad con la herramienta MakeAppx.exe para MSIX | Puede usar la herramienta MakeAppx.exe para crear un paquete MSIX para aplicaciones UWP o aplicaciones de escritorio tradicionales. Esta herramienta se incluye en el SDK de Windows 10 y puede usarse desde un símbolo del sistema o un archivo de script. </br> Las aplicaciones UWP, consulte [crear un paquete de aplicación con la herramienta MakeAppx.exe](https://docs.microsoft.com/windows/uwp/packaging/create-app-package-with-makeappx-tool). </br> Para aplicaciones de escritorio, consulte [empaquetar manualmente una aplicación de escritorio](https://docs.microsoft.com/windows/uwp/porting/desktop-to-uwp-manual-conversion).
Marco de compatibilidad de paquete | El [el marco de trabajo de paquete de soporte técnico](https://docs.microsoft.com/windows/msix/package-support-framework-overview) es un kit de código abierto que ayuda a se aplican correcciones a la aplicación de escritorio existente cuando no tienes acceso al código fuente, para que se pueda ejecutar en un contenedor MSIX.
