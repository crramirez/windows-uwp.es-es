# <a name="whats-new-in-windows-10-for-developers-build-17763"></a>Novedades de Windows 10 para los desarrolladores, compilación 17763

17763 de compilación de Windows 10 (también conocida como la de 2018 octubre Update o versión 1809), en combinación con Visual Studio 2017 y el SDK actualizado, proporciona las herramientas, características y experiencias para crear aplicaciones extraordinarias para la plataforma Universal de Windows. [Instala las herramientas y el SDK](http://go.microsoft.com/fwlink/?LinkId=821431) en Windows 10 y estarás listo para [crear una nueva aplicación universal de Windows](../get-started/create-uwp-apps.md) o para aprender a usar el [código de aplicación existente en Windows](../porting/index.md).

A continuación te ofrecemos una colección de características e instrucciones nuevas y mejoradas de interés para los desarrolladores de Windows en esta versión. Para obtener una lista completa de los nuevos espacios de nombres agregado a Windows SDK, consulta la [17763 cambios en la API de la compilación de Windows 10](windows-10-build-17763-api-diff.md). Para obtener más información sobre las características más destacadas de Windows 10, consulta [Lo bueno de Windows 10](http://go.microsoft.com/fwlink/?LinkId=823181). También puedes consultar [Funcionalidades de la plataforma de desarrolladores de Windows](https://developer.microsoft.com/windows/platform/features) si quieres tener una descripción general de alto nivel de las funcionalidades pasadas y futuras que se han agregado en la plataforma de Windows.

## <a name="samples"></a>Ejemplos

### <a name="customer-orders-database"></a>Base de datos de pedidos de clientes

La [muestra de base de datos de pedidos de clientes](https://github.com/Microsoft/Windows-appsample-customers-orders-database) se ha actualizado para usar controles nuevos, como [DataGrid](https://docs.microsoft.com/windows/communitytoolkit/controls/datagrid), [NavigationView](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.navigationview)y [un ampliador](https://docs.microsoft.com/windows/communitytoolkit/controls/expander).

### <a name="customer-database-tutorial"></a>Tutorial de base de datos de cliente

[Tutorial de base de datos de cliente](../enterprise/customer-database-tutorial.md) crea una aplicación básica de UWP para la administración de una lista de clientes y presenta los conceptos y procedimientos recomendados útiles en el desarrollo empresarial. Te guiará a través de implementar los elementos de la interfaz de usuario y agregar las operaciones en una base de datos SQLite local y se proporcionan instrucciones sueltos para la conexión a una base de datos REST remota si lo deseas ir más allá.

### <a name="photo-editor-cwinrt"></a>Photo Editor C++ / WinRT

La [aplicación de muestra de Photo Editor](https://github.com/Microsoft/Windows-appsample-photo-editor) muestra el desarrollo con la [C++ / WinRT](../cpp-and-winrt-apis/intro-to-using-cpp-with-winrt.md) proyección de lenguaje. La aplicación te permite recuperar fotos desde la biblioteca de **imágenes** y, a continuación, modificar una imagen seleccionada con efectos fotográficos asociado.

### <a name="windows-machine-learning"></a>Aprendizaje automático de Windows

El repositorio de [Aprendizaje automático de Windows](https://github.com/Microsoft/Windows-Machine-Learning) se ha actualizado para trabajar con el SDK más reciente de Windows 10 y contiene ejemplos escritos en C#, C++ y JavaScript.

### <a name="xaml-hosting-api"></a>API de hospedaje de XAML

La [muestra de la API de hospedaje de XAML](https://github.com/Microsoft/Windows-appsample-Xaml-Hosting) es una aplicación de escritorio de Win32 que resalta los distintos escenarios mediante la API (también denominado Islas XAML) de hospedaje de XAML de UWP. El proyecto incorpora controles de entrada de lápiz de Windows, el Reproductor multimedia y la vista de navegación en una presentación de estilo de la galería. Uso de controles de fuera de general, la muestra también muestra cómo controlar XAML y eventos de Windows nativo y mensajes y enlace de datos básico de XAML.
