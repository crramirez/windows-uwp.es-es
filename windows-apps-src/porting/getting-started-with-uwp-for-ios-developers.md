# <a name="getting-started-with-uwp-for-ios-developers"></a>Introducción a UWP para desarrolladores de iOS


Si es un desarrollador de iOS que está pensando en desarrollar para Windows 10, estos documentos son un buen punto de partida. Le presentarán algunos de los conceptos que necesitará saber a medida que empiece a escribir aplicaciones, y se explicará cómo puede publicar su trabajo en el Microsoft Store.

En esta sección, empezaremos gradualmente y examinaremos cómo usar Microsoft Visual Studio y el lenguaje de programación C# para crear una aplicación de prueba y, en concreto, explicaremos cómo puede variar el proceso respecto de las herramientas que usas actualmente. (¿No te gusta C#? No te preocupes, hay otros lenguajes de programación y herramientas, que trataremos en [Introducción: elección de un lenguaje de programación](getting-started-choosing-a-programming-language.md)).

Windows 10 presenta una nueva plataforma para crear aplicaciones atractivas en equipos de escritorio, portátiles, tabletas, teléfonos y mucho más. Como las aplicaciones de la plataforma universal de Windows (UWP) proporcionan numerosas funcionalidades exclusivas, una migración directa de tu aplicación de iOS te impedirá usarlas. En consecuencia, te animamos a que pruebes los nuevos controles y habilidades para que veas cómo pueden facilitarte tu labor como desarrollador y permitirte crear nuevas aplicaciones.

En conclusión: no te limites a migrar tu aplicación, **reimagina** tu aplicación y aprovecha las nuevas funcionalidades y dispositivos. No se liquide para el denominador común más bajo, sino también para crear experiencias enriquecidas que usan características únicas de Windows 10, como iconos dinámicos, notificaciones e interacción con Cortana.

Para empezar a trabajar con estos tutoriales, necesitará un equipo con Windows 10 y Microsoft Visual Studio instalado. Puede descargarlos desde las [descargas para desarrolladores para programar aplicaciones para UWP](https://developer.microsoft.com/windows/downloads). ¿No tienes PC? No te preocupes, puedes usar tu Mac. Consulta [Instalar Windows y las herramientas de desarrollador en un Mac](setting-up-your-mac-with-windows-10.md).
