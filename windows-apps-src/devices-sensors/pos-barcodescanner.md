# <a name="barcode-scanner"></a>Escáner de códigos de barras

Esta sección proporciona instrucciones para crear aplicaciones de la Plataforma universal de Windows (UWP) que usan un escáner de código de barras.

|Tema |Descripción |
|------|------------|

| [Uso de un desencadenador de software](../devices-sensors/pos-barcodescanner-software-trigger.md) | Obtenga información sobre cómo controlar el acto de análisis de software. |

| [Obtener y comprender los datos de código de barras](pos-barcodescanner-scan-data.md) | Obtenga información sobre cómo obtener e interpretar los datos de código de barras que examinan. |
| [Escáner de cámara](pos-camerabarcode.md) | Lee códigos de barras a través de una lente de cámara estándar desde una aplicación de la Plataforma universal de Windows. Requiere la Actualización de abril de 2018 de Windows 10 (compilación 17134 o posterior). |
