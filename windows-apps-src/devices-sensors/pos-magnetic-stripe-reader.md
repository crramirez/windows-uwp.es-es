# <a name="magnetic-stripe-reader"></a>Lector de bandas magnéticas

Un lector de bandas magnéticas puede examinar determinados tipos de tarjetas que tienen un bandas magnéticas en ellos, tales como tarjetas de crédito, las licencias del controlador y almacenan tarjetas de fidelización. Es probable que haya encontrado uno al comprar algo en una tienda.

Es similar al uso de cualquier otro dispositivo de punto de servicio, usando un lector de bandas magnéticas en la aplicación de plataforma Universal de Windows (UWP) en que se deben enumerar los dispositivos disponibles, crea el objeto de dispositivo y reclamar el dispositivo antes de poder usar. Sigue la documentación en [Introducción al punto de servicio](pos-basics.md) para obtener más información.
