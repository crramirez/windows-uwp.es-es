# <a name="latest-updates-to-the-windows-developer-docs"></a>Actualizaciones más recientes de los documentos para desarrolladores de Windows

Los documentos para desarrolladores de Windows se actualizan periódicamente con datos y contenido nuevos y mejorados. A continuación se muestra un resumen de los cambios al 5 de octubre de 2020.

Nota: Para obtener una lista específica de las API agregadas como parte de la compilación 19041 de Windows 10 (también denominada 2004), consulte [esta lista](/windows/uwp/whats-new/windows-10-build-19041-api-diff).


## <a name="news"></a>Novedades


Este mes hemos traducido otro lote de vínculos absolutos a vínculos relativos para acelerar la navegación y mejorar la experiencia de lectura sin conexión. Continúe para indicarnos si encuentra vínculos rotos: puede proporcionar comentarios directamente desde la mayoría de las páginas, o puede ponerse en contacto con nosotros en Twitter con nuestro identificador [@WindowsDocs](https://www.twitter.com/windowsdocs).

* Documentación actualizada:

    * Actualizaciones de la [documentación de Surface Duo](https://docs.microsoft.com/dual-screen/introduction)
