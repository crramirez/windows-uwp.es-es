# <a name="latest-updates-to-the-windows-developer-docs"></a>Actualizaciones más recientes de los documentos para desarrolladores de Windows

Los documentos para desarrolladores de Windows se actualizan periódicamente con datos y contenido nuevos y mejorados. A continuación se muestra un resumen de los cambios a partir del 3 de septiembre de 2020.

Nota: Para obtener una lista específica de las API agregadas como parte de la compilación 19041 de Windows 10 (también denominada 2004), consulte [esta lista](/windows/uwp/whats-new/windows-10-build-19041-api-diff).


## <a name="news"></a>Novedades

Este mes hemos traducido varios miles de vínculos absolutos a vínculos relativos para acelerar la navegación y mejorar la experiencia del libro sin conexión. Si encuentra vínculos rotos, háganoslo saber. Puede proporcionar comentarios desde la mayoría de las páginas o ponerse en contacto con nosotros en Twitter con nuestro identificador **WindowsDocs**.

Los aspectos destacados de este mes incluyen:

* Actualizaciones en una serie de tutoriales de cuatro partes para usar la [interfaz de usuario](/windows/uwp/design/basics/xaml-basics-ui) y el [enlace de datos](/windows/uwp/data-binding/xaml-basics-data-binding) de WinUI, así como el [código de ejemplo](https://github.com/microsoft/Windows-appsample-photo-lab/tree/master/xaml-basics-starting-points). 

* Actualizaciones de la documentación de [Windows Terminal 1.2](/windows/terminal/) y Preview 1.3.

* Nueva página [Configurar el entorno de desarrollo](../../dev-environment/overview.md).

* Nuevas actualizaciones de las preguntas más frecuentes sobre [inteligencia artificial y aprendizaje automático](/windows/win32/direct3d12/gpu-faq).

* Nuevo tema de WinRT/C++ sobre [Asincronía e interoperabilidad entre C++/WinRT y C++/CX](/windows/uwp/cpp-and-winrt-apis/interop-winrt-cx-async).
